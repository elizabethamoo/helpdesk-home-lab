# Screenshots

This folder contains screenshots from the ServiceNow practice environment.

## What to Screenshot

### ServiceNow Dashboard
- Main dashboard showing open and closed ticket counts
- Ticket list view showing multiple tickets
- Individual ticket detail for TKT-001, TKT-002, and TKT-003

### Azure AD / Microsoft Entra ID
- User list view in Azure AD Admin Center
- User account properties page
- Password reset screen
- MFA enrollment screen

### CLI Diagnostic Outputs
- ipconfig /all output in Command Prompt
- Ping test success and failure examples
- Tracert output
- Nslookup output

## Naming Convention

Use lowercase with hyphens:
- servicenow-dashboard.png
- ticket-001-detail.png
- azure-ad-password-reset.png
- ipconfig-output.png
- ping-test-result.png
